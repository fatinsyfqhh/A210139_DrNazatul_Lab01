
### Lab Description
This lab 1 exercises the implementation of modern declarative user interfaces on the Android platform. The primary focus of this assignment is learning how to build and structure UI layouts using **Jetpack Compose** components, managing structural component arrangements and applying cohesive styles using Material Design guidelines.

### Implementation Features
* Core layout construction using Composable functions (`@Composable`).
* UI element hierarchy arrangement using structured `Row`, `Column` and `Box` layouts.
* Native custom text fields, image holders and arrangement modifiers.
* Implementation of explicit element spacing, structural paddings and sizing controls.
* Visual presentation compliance with standard Material Design rules.

### Technologies Used
* Kotlin
* Jetpack Compose
* Material Design 3 Components
* Android Studio

### Setup Instructions
1. Clone this repository to your local system.
2. Open the project folder in **Android Studio**.
3. Allow Gradle to synchronize dependencies automatically.
4. Run the application using an active Android Virtual Device (AVD) emulator or via a physical developer device connected through USB debugging.
